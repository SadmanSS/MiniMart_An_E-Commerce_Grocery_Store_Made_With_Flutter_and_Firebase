MiniMart is a modern, feature-rich e-commerce mobile application built with Flutter. It provides a seamless shopping experience with user authentication, product browsing, cart management, and order tracking. The app leverages Firebase for real-time data and authentication, ensuring a robust and scalable backend.

## ‚ú® Features

- **User Authentication**: Secure login and signup using Firebase Auth.
- **Dynamic Product Showcase**: Browse products by categories with rich images and details.
- **Smart Cart System**: Add/remove items, adjust quantities, and view total cost in real-time.
- **Order Management**: Place orders and track order history.
- **Profile Customization**: Manage user profile details including name, phone, and location.
- **Location Selector**: Choose your delivery location for localized service.
- **Optimized Performance**: Efficient image caching and state management using Provider.
- **Modern UI/UX**: Sleek design with custom widgets, gradients, and intuitive navigation.

## üõ†Ô∏è Tech Stack

- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Firestore, Authentication)
- **State Management**: Provider
- **Image Caching**: cached_network_image, flutter_cache_manager

## üöÄ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/minimart.git
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Create a project in the [Firebase Console](https://console.firebase.google.com/).
   - Add an Android/iOS app to your Firebase project.
   - Download the `google-services.json` (for Android) and place it in `android/app/`.
   - Download the `GoogleService-Info.plist` (for iOS) and place it in `ios/Runner/`.

4. **Run the app**
   ```bash
   flutter run
   ```
